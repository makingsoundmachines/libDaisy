Detailed Description

Driver for ISSI IS31FL3731 144 ch PWM LED Matrix Ported from the Adafruit IS31FL3731 library - https://github.com/adafruit/Adafruit_IS31FL3731/ Port for Daisy by Making Sound Machines - https://github.com/makingsoundmachines

LED driver for one or multiple ISSI IS31FL3731 144ch 8bit PWM chips connected to a single I2C peripheral. This driver uses two buffers - one for drawing, one for transmitting. Multiple Is31fl3731 instances can be used at the same time.
