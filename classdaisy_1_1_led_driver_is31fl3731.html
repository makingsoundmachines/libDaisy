Detailed Description

template<int Is31fl3731_numDrivers, bool Is31fl3731_persistentBufferContents = true>
class daisy::LedDriverIs31fl3731< Is31fl3731_numDrivers, Is31fl3731_persistentBufferContents >

LED driver for one or multiple Is31fl3731 8it PWM chips connected to a single I2C peripheral. It includes gamma correction from 8bit brightness values but it can also be supplied with raw 8bit values. This driver uses two buffers - one for drawing, one for transmitting. Multiple LedDriverIs31fl3731 instances can be used at the same time.

Parameters
Is31fl3731_numDriversThe number of Is31fl3731 driver attached to the I2C peripheral.
Is31fl3731_persistentBufferContentsIf set to true, the current draw buffer contents will be copied to the next draw buffer during SwapBuffersAndTransmit(). Use this, if you plan to write single leds at a time. If you will alway update all leds before calling SwapBuffersAndTransmit(), you can set this to false and safe some cycles.
